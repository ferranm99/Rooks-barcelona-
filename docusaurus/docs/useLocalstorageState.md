## About

UseState but auto updates values to localStorage

[//]: # "Main"

## Installation

    npm install --save @rooks/useLocalstorageState

## Importing the hook

```javascript
import useLocalstorageState from "@rooks/useLocalstorageState"
```

## Usage

```jsx
function Demo() {
  useLocalstorageState();
  return null
}

render(<Demo/>)
```
