## About
Very similar useKey but it returns a ref


[//]: # (Main)

## Installation

```
npm install --save @rooks/use-key-ref
```

## Importing the hook

```javascript
import useKeyRef from "@rooks/use-key-ref"
```

## Usage

```jsx
function Demo() {
  useKeyRef();
  return null
}

render(<Demo/>)
```
